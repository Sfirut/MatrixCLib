# Пользовательская документация  

## 1. Введение  

Библиотека предоставляет набор функций для работы с двумерными матрицами на языке C. Она поддерживает все основные операции над матрицами, включая создание, копирование, арифметические операции, сортировку и прочие преобразования.  

### Структура данных  

Матрицы представлены как `float**` (двойной указатель на `float`), где:  
- Первая размерность - строки  
- Вторая размерность - столбцы  

---

## 2. Начало работы  

### 2.1 Подключение библиотеки  

Добавьте в ваш проект заголовочный файл:  
```c
#include "matrix.h"
```

### 2.2 Сборка программы  

При компиляции убедитесь, что файлы `matrix.c` и `matrix.h` доступны компилятору:  
```bash
gcc your_program.c matrix.c
```

# Документация по тестированию  

## 1. Введение  

Тестирование библиотеки включает проверку:  
- Корректности работы функций.  
- Обработки ошибок (неверные размеры, нулевые указатели).  
- Граничных случаев (пустые матрицы, особые значения).  

## 2. Тестплан  

### 2.1. Базовые операции  

| **Тест-кейс**                  | **Ожидаемый результат**                     |
|--------------------------------|---------------------------------------------|
| Сложение двух матриц           | Корректная сумма элементов                 |
| Умножение матриц               | Правильное произведение                    |
| Транспонирование               | Столбцы становятся строками                |

### 2.2. Обработка ошибок  

| **Тест-кейс**                  | **Ожидаемый результат**                     |
|--------------------------------|---------------------------------------------|
| Передача указателя NULL в функции | Возврат ошибки                            |
| Некорректные размеры матриц    | Возврат ошибки                             |

### 2.3. Особые случаи  

| **Тест-кейс**                  | **Ожидаемый результат**                     |
|--------------------------------|---------------------------------------------|
| Матрица 1x1                    | Корректная работа                          |
| Нулевая матрица                | Корректная обработка                       |

## 3. Основные этапы тестирования  

1. **Создание тестовых данных:**  
   - Матрицы с известными значениями.  
   - Матрицы с нулевыми и отрицательными значениями.  

2. **Запуск тестов:**  
   - Проверка возвращаемых значений.  
   - Визуальный контроль ввода (с помощью функции `matrix_print`).  

3. **Проверка и анализ ошибок:**  
   - Передача некорректных аргументов.  
   - Проверка корректности сообщений об ошибках.
